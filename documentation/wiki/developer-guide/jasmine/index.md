# JAS-mine Architecture

SimPaths is built on the [JAS-mine](https://www.microsimulation.ac.uk/jas-mine/) simulation libraries. Understanding the JAS-mine architecture is essential for developing or modifying SimPaths.

New developers are recommended to start with the following external resources:

- [JAS-mine GitHub repository](https://github.com/jasmineRepo)
- [JAS-mine core API](https://raw.githack.com/jasmineRepo/JAS-mine-core/master/microsim-core/doc/index.html)
- [JAS-mine documentation](https://www.microsimulation.ac.uk/jas-mine/)
